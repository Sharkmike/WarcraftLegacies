Class ArtifactManager

Manages all Artifacts by maintaining a list of registered Artifacts indexed by item type, and by firing Artifact related events.

When a hero carrying an Artifact dies, the Artifact is dropped to the floor, or to the nearest Shore if the hero died on the water.
